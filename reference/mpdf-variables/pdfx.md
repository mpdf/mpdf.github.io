(mPDF >= 5.1)

PDFX – Create PDF/X-1a compliant document

# Description

void **PDFX**

Specify whether to create a PDF/X-1a compliant document.

# Values

<span class="parameter">$PDFX</span> =  `true`\|`false`

**Values**

* `true`: create PDF/X-1a compliant document   
* `false`: normal PDF document

Default: `false`
