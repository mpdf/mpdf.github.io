float **$annotOpacity** = `0.5`    

Default Opacity for Annotations;

**Value** 

* >`0` to <= `1`

Suggest `0.5` if using in-line with text, or `1` if using in the margins (with annotMargin &lt; &gt; `0`)

Default: `0.5`

# Usage

Set at document initiation
```php
<?php
$mpdf = new \Mpdf\Mpdf([
    'annotMargin' => 5,
    'annotOpacity' => 1
]);

```
