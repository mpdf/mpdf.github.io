Upgrading 4.3 to 4.4 - Archived

If you have made changes to any of your configuration files etc. please keep a safe copy first. These may include:

  • config.php
  • config_cp.php
  • config_fonts.php
  • mpdf.css

Download the Upgrade .zip file and unzip it

  • Upload all of the included files to the server, maintaining the folders as they are.
Backwards Compatibility

All the changes in 4.4 are backwards compatible except the handling of some background-images (which are no longer resized if they are too large to fit).
