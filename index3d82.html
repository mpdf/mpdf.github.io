Upgrading 4.2 to 4.3 - Archived

If you have made changes to any of your configuration files etc. please keep a safe copy first. These may include:

  • config.php
  • config_cp.php
  • config_fonts.php
  • mpdf.css

Download the Upgrade .zip file and unzip it

  • Upload all of the included files to the server, maintaining the folders as they are
  • You can optionally delete the mpdfi/mpdfi.php file as it is no longer used
Backwards Compatibility

All the changes in 4.3 are backwards compatible except the use of mPDFI (importing PDF files or templates). You will need to make minor changes to your scripts.
See SetImportUse() for more information.
