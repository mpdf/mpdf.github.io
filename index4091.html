Note: AddPages() is now depracated in favour of AddPage(). The function was added in mPDF 1.3 with the optional values of "NEXT-ODD" and "NEXT-EVEN" for the parameter type. From mPDF 2.2 these values were added to AddPage(), giving it the same functionality. Older scripts using AddPages() can be simply changed to AddPage(), leaving all parameters unchanged.

Note: This function was removed completely in mPDF v 6.0
