Folder for temporary files

mPDF is configured to use [your_path_to_mpdf]/tmp/ as a folder to write temporary files (mainly for images). Write permissions must be set on this folder to allow read/write access for the script.

If you wish to use a different folder for temporary files, you should define the constant _MPDF_TEMP_PATH before including the mpdf.php file e.g.

define("_MPDF_TEMP_PATH", '../../common/tempfiles/');

include("../mpdf.php");

$mpdf=new mPDF();
