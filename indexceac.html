useOnlyCoreFonts - Archived

Note: DEPRACATED from mPDF >= 5.0  Now uses $mpdf = new mPDF('c');

Description

boolean useOnlyCoreFonts

When TRUE, forces mPDF to use the core Adobe fonts (Helvetica, Times and Courier), thus keeping the PDF file size to a minimum. Thus a CSS stylesheet defining Verdana will be substituted by Helvetica, Georgia by Times etc.

Note: Ignored if mPDF is using any codepage other than win-1252.

Values

useOnlyCoreFonts = TRUE|FALSE

Values
TRUE: Use only core Adobe system fonts (if using codepage="win-1252")
FALSE: DEFAULT Use all available fonts

Examples

Example #1

<?php

$mpdf=new mPDF('win-1252');
$mpdf->useOnlyCoreFonts = true;
$mpdf->WriteHTML('<p style="font-family: Verdana">Hallo World</p>');
$mpdf->Output('filename.pdf');

?>

Will output Hallo World in Helvetica font (Adobe system font)

Notes

Note: This variable was renamed from use_embeddedfonts_1252 in mPDF 3.0 as it more accurately describes its function.
