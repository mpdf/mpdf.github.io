list_indent_default

(mPDF >= 6.0)

Description

void list_indent_default

Define the default indentation of a list item, when in standard 'browser' list mode.

For more information, see Lists.

Values

list_indent_default

Values
LENGTH: Any valid CSS length value is permitted e.g. "10pt" or "3em"
DEFAULT: '40px'
