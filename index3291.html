rtlCSS - Archived

Note: Removed in mPDF v5.1

$mpdf->rtlCSS = 2;
When using RTL (right-to-left) languages e.g. arabic, mPDF automatically reverses the alignment for paragraphs, table cells etc. Set to a value 0, 1 or 2 determines at what level the CSS styles are overridden. This is determined by whether the HTML code or CSS stylesheet is generated specifically for the language being used.
2 (default) - mPDF will reverse all alignment, whether set in stylesheets or defined in-line (e.g. <p style="text-align:right">)
1 - reverses only stylesheets, observing correctly any in-line styles
0 - reverses only the default settings (set in $default_CSS as the top of mpdf.php file), correctly observing any stylesheet definitions
Default = 2
