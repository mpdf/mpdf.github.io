tableMinSizePriority

(mPDF >= 4.6)

Description

mixed tableMinSizePriority

Forces mPDF to respect the value set as autosize when displaying a table. If a table has page-break-inside:avoid set but cannot be made to fit onto a page without exceeding the resizing factor set by autosize: the value of tableMinSizePriority will determins which factor wins out .

Values

tableMinSizePriority

Values
TRUE | FALSE
DEFAULT = FALSE
