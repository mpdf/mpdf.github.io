Font substitution

(mPDF <= 4.x)

If a font or default font is declared in mPDF (including the default stylesheet, and any CSS stylesheets etc.) the font that is actually set is determined by:

UTF-8 codepage

The font requested (including style) is checked to see if it is available to mPDF: set by the array available_unifonts* defined in mpdf_config.php
If the requested font (and style) is not available:

  1. If the font[nostyle] exists - selected
  2. If only one font available in available_fonts* - selected
  3. Substitute a font of similar type (sans-serif, serif, or mono) - this first one in the sans_fonts, serif_fonts, and mono_fonts arrays are used. The font-family-type is determined by the three arrays sans_fonts, serif_fonts, and mono_fonts defined in mpdf_config.php
  4. If no font has yet been selected, the first font in the array available_fonts* is selected

* Note that the array available_unifonts is altered when certain languages are defined e.g.
If 'ur' (Urdu) is set as the language, the only font which contains the necessary characters is Scheherazade. In config_cp.php, the value $mpdf_pdf_unifonts = "scheherazade"; is used to redefine array available_unifonts to only include Scheherazade, and thus restrict font selection.

CJK codepages

If use_CJK_only is set to true (default=false, see Getting Started), then only the fonts big5, sjis, uhc, and gb can be selected. All other fonts are ignored.

Otherwise, any of the CJK fonts or other Unicode fonts can be selected (as above).

win-1252 codepage

If useOnlyCoreFonts is set to true (default=false, see Getting Started), then all font requests will be substituted by Arial/Helvetica, Times or Courier i.e. the core Adobe fonts. mPDF determines whether the requested font is a sans-serif, serif or monospace font, and substitutes accordingly. The font-family-type is determined by the three arrays sans_fonts, serif_fonts, and mono_fonts defined in mpdf_config.php

If useOnlyCoreFonts is set to false (default), then fonts are checked to see if they are available to mPDF: set by the array available_fonts defined in mpdf_config.php
If the requested font (and style) is not available:

  1. If the font[nostyle] exists - selected
  2. If only one font available in available_fonts - selected
  3. Substitute a font of similar type (sans-serif, serif, or mono) - this first one in the sans_fonts, serif_fonts, and mono_fonts arrays are used
  4. If no font has yet been selected, the first font in the array available_fonts is selected
Other codepages

As for win-1252 when useOnlyCoreFonts is set to false.

Character (font) substitution

If some circumstances, individual characters are replaced by glyphs from another font - see character (font) substitution.
