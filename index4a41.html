Installation v4.x - Archived

First-time users

Installation:

  • Download the .zip file and unzip it
  • Create a folder e.g. /mpdf on your server
  • Upload all of the files to the server, maintaining the folders as they are
  • Ensure that you have write permissions set for the /tmp/ and for /graph_cache/
  • If you wish to use embedded font subsets, please see the notes for further instructions.

If you wish to install additional fonts for Indic, Arabic or CJK languages, please see the notes for further instructions.

To test the installation, point your browser to one of the basic example files e.g. [path_to_mpdf_folder]/mpdf/examples/example01_basic.php

If you wish to define a different folder for temporary files rather than /tmp/ see the note on 'Folder for temporary files'.

If you have problems, please read the section on troubleshooting in the manual.


Upgrading from versions of mPDF < 4

There have been a lot of changes in version 4. The whole structure of the package was changed, and a number of fonts updated etc.

I strongly recommend removing previous versions from your server completely, and performing a clean installation of version 4. One good reason for doing this is that some of the fonts have been renamed with lowercase names (e.g. Freeserif.z is now freeserif.z); some FTP agents to upload files to servers will not change the file name because it thinks Filename == filename.

Installation: follow the instructions as for a new user, but if you have made changes to any of your configuration files etc. please keep a safe copy first. These may include:

  • mpdf_config.php
  • mpdf.css
  • htmltoolkit.php
  • mpdf.php
  • any font files you have added

If you wish to use embedded font subsets, please see the notes for more information and further instructions.

If you wish to install additional fonts for Indic, Arabic or CJK languages, please see the notes for more information and further instructions.

If you wish to define a different folder for temporary files rather than /tmp/ see the note on 'Folder for temporary files'.
