Upgrading 4.4 to 4.5 - Archived

If you have made changes to any of your configuration files etc. please keep a safe copy first. These may include:

  • config.php
  • config_cp.php
  • config_fonts.php
  • mpdf.css

Download the Upgrade .zip file and unzip it

  • Upload all of the included files to the server, maintaining the folders as they are.
Backwards Compatibility

All the changes in 4.5 are backwards compatible except the setting of the configurable variable $this->watermarkImgBehind in config.php.

Fonts

Some minor bugs were fixed in the utility to make font files to embed subsets. Despite this, the font files seem to function perfectly OK. The font files offered with mPDF have been re-generated, so you can download the new ones if you are having problems. Otherwise, it is probably OK to leave alone.
