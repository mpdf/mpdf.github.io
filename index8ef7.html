Reversed RTL text - Archived

RTL text in Bookmarks and metadata was not reversed when viewing with Adobe Reader prior to version 9.

mPDF < 3.0 reversed RTL text (Bookmarks and metadata) so that it displayed correctly.

 

From mPDF >= 3.0, mPDF no longer reverses RTL text (in Bookmarks and metadata) so that it will correctly appear in Adobe Reader version 9.

(Metadata includes Title, Creator, Author, Keywords and Subject.)
