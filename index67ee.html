Description

<toc />

 

Note: <toc> had to be used at the end of the document in early versions of mPDF to generate the Table of Contents. This is now generated automatically when Output() is called, and <toc> should no longer be used (mPDF >= 2.0). It is now recommended that you use TOCpagebreak() or <tocpagebreak> and TOC_Entry() or <tocentry>.
